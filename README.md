# Time Manager

Web app created with React and Vite and written in TypeScript.

Add more structure to your day by visualizing time spent on daily activities with the use stopwatches and timers. Balance productive activities with leasure by earning time with stopwatches and spending time with timers. The app will notify the user when a timer reaches zero.

Example use case: Two stopwatches, one named "Programming" and one named "Sensible", and a timer named "Gaming". Max 4 hours of gaming daily and until no later than 22 PM.

Designed for personal use with flexibility for other use cases, and developed to improve web development skills.
